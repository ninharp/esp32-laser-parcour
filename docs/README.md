# Documentation

This directory contains additional documentation for the ESP32 Laser Obstacle Course project.

## Structure

- `hardware/` - Circuit schematics, PCB designs, and wiring diagrams
- `api/` - API documentation and component interfaces
- `images/` - Photos, diagrams, and illustrations

## Contributing Documentation

When adding new documentation:

1. Use clear, descriptive filenames
2. Include circuit diagrams in both PDF and editable format (e.g., KiCad)
3. Add photos showing actual hardware setup
4. Document any modifications or improvements
